# Лабораторная работа 4: Поведенческие и структурные паттерны

## Описание работы
Реализация 6 паттернов проектирования на языке Python в рамках лабораторной работы по курсу "Технологии программирования".

## Структура проекта
strategy.py # Стратегия - выбор оружия
chain_of_responsibility.py # Цепочка - техподдержка
iterator.py # Итератор - инвентарь
proxy.py # Прокси - защита сервера
bridge.py # Мост - графика игры
adapter.py # Адаптер - системы измерений
test_patterns.py # Вайл с тестами
